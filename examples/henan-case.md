# 河南案例：传统高竞争省份，保本科与就业路径并重

> 示例为产品演示稿，不引用真实 2026 招生数据，不构成填报建议。正式填报必须以河南省教育考试院和高校官方招生计划为准。

## 1. 用户输入

| 字段 | 内容 |
| --- | --- |
| 省份 | 河南 |
| 年份 | 2026 |
| 批次 | 本科批 |
| 科类/选科 | 物理类，物化政 |
| 分数/位次 | 548 分，约 82000 名 |
| 预算 | 公办优先，民办可作为兜底但费用要可控 |
| 地域 | 河南省内优先，湖北、安徽、河北、山西可接受 |
| 专业偏好 | 电气、自动化、机械、计算机、师范 |
| 排斥专业 | 土木、化工、材料、环境、生物、护理 |
| 路径期望 | 保本科、就业稳定、考研或考编 |
| 风险偏好 | 保守，不能为了城市牺牲专业下限 |

## 2. 看板总览

| 模块 | 状态 | 一句话判断 |
| --- | --- | --- |
| 录取安全 | 黄灯 | 能做本科方案，但保底必须足够深。 |
| 专业匹配 | 黄灯 | 电气/自动化/机械比计算机更适合作主线。 |
| 地域偏好 | 黄灯 | 只看省内容易拥挤，应加周边省份对冲。 |
| 成本风险 | 黄灯 | 民办可兜底，但要防高费用低质量。 |
| 最大风险 | 红灯 | 保底不真导致滑档或被迫接受差专业。 |

## 3. 候选池分层

| 层级 | 候选方向 | 用途 | 进入条件 | 风险灯 |
| --- | --- | --- | --- | --- |
| 冲 | 省内较好公办工科组 | 少量上行 | 位次接近且专业下限可接受 | 黄/红 |
| 稳 | 周边省份公办应用工科 | 主力组合 | 位次有缓冲，专业可接受 | 绿/黄 |
| 保 | 省内/周边低热公办本科 | 保本科 | 组内专业能接受 | 绿 |
| 兜底 | 费用可控民办强应用专业 | 防止滑档 | 学费、办学质量、专业都可接受 | 黄 |
| 剔除 | 高价民办泛管理、弱就业专业 | 避坑 | 成本高、路径弱 | 红 |

## 4. 专业路径

| 专业方向 | 匹配度 | 出路主线 | 主要坑点 | 建议 |
| --- | --- | --- | --- | --- |
| 电气 | 高 | 电力、能源、制造、考研 | 学校平台影响通道 | 主线 |
| 自动化 | 中-高 | 制造、控制、装备、考研 | 名称宽泛，需看课程 | 主线 |
| 机械 | 中 | 制造、设备、车辆、考研 | 工作环境和薪资预期要现实 | 保底主线 |
| 计算机 | 中 | 软件、考研、应用开发 | 热度高，普通校分化 | 少量冲，不做唯一主轴 |
| 师范 | 中 | 考编、教育、考研 | 地方编制竞争 | 备选 |

## 5. 风险热力图

| 风险项 | 等级 | 影响 | 处理 |
| --- | --- | --- | --- |
| 保底深度不足 | 高 | 滑档风险 | 增加深保和兜底 |
| 省内竞争拥挤 | 中 | 同分段选择变差 | 加周边省份对冲 |
| 民办费用和质量 | 高 | 成本高且回报弱 | 单独审计，不默认接受 |
| 专业下限不可接受 | 高 | 入学后执行风险 | 全组审计，不合格剔除 |
