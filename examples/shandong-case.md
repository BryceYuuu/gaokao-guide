# 山东案例：3+3 综合，专业 + 院校模式下的就业稳健方案

> 示例为产品演示稿，不引用真实 2026 招生数据，不构成填报建议。正式填报必须以山东省教育招生考试院和高校官方招生计划为准。

## 1. 用户输入

| 字段 | 内容 |
| --- | --- |
| 省份 | 山东 |
| 年份 | 2026 |
| 批次 | 普通类常规批 |
| 科类/选科 | 3+3 综合，物理 + 化学 + 地理 |
| 分数/位次 | 575 分，约 63000 名 |
| 预算 | 公办优先，民办谨慎，中外合作需低于 5 万/年 |
| 地域 | 山东省内优先，华东/华北可接受 |
| 专业偏好 | 电气、自动化、电子信息、计算机、师范数学 |
| 排斥专业 | 土木、环境、生物、旅游管理、市场营销 |
| 路径期望 | 直接就业、考研、国企央企、教师编制可考虑 |
| 风险偏好 | 保守稳妥，专业必须能接受 |

## 2. 看板总览

| 模块 | 状态 | 一句话判断 |
| --- | --- | --- |
| 录取安全 | 黄灯 | 专业 + 院校模式更透明，但热门工科仍需拉开梯度。 |
| 专业匹配 | 绿灯 | 物化基础可支撑工科主线。 |
| 地域偏好 | 黄灯 | 省内优先可保留，但应加入华东/华北对冲。 |
| 专业风险 | 黄灯 | 计算机热度高，电气/自动化更适合作主线。 |
| 最大风险 | 红灯 | 用热门专业挤掉真正可接受的稳保项。 |

## 3. 候选池分层

| 层级 | 候选方向 | 用途 | 进入条件 | 风险灯 |
| --- | --- | --- | --- | --- |
| 冲 | 省内热门公办信息类专业 | 上行机会 | 位次接近且计划数不小 | 黄/红 |
| 稳 | 省内/周边电气、自动化、电子信息 | 主力组合 | 近年位次有缓冲 | 绿/黄 |
| 保 | 地市公办院校可接受工科/师范专业 | 防滑档 | 专业本身可接受 | 绿 |
| 观察 | 师范数学、行业特色应用工科 | 路径备选 | 核验就业/考编/考研数据 | 灰/黄 |
| 剔除 | 高价中外合作或弱民办泛管理专业 | 避坑 | 费用/质量/就业不匹配 | 红 |

## 4. 专业路径

| 专业方向 | 匹配度 | 出路主线 | 主要坑点 | 建议 |
| --- | --- | --- | --- | --- |
| 电气 | 高 | 电力、能源、制造、考研 | 地方电网通道需证据 | 主线 |
| 自动化 | 高 | 智能制造、控制、考研 | 课程平台差异大 | 主线 |
| 电子信息 | 中-高 | 通信、硬件、制造 | 强校和普通校差距大 | 主线备选 |
| 计算机 | 中 | 软件、考研、外包/中小厂 | 热度和就业分化 | 少量冲稳，不做唯一主轴 |
| 师范数学 | 中 | 考编、教育、考研 | 地方编制和学科需求变化 | 备选 |

## 5. 风险热力图

| 风险项 | 等级 | 影响 | 处理 |
| --- | --- | --- | --- |
| 热门专业位次上移 | 高 | 冲稳误判 | 用近三年同专业位次做区间 |
| 计划数过少 | 中 | 波动放大 | 小计划数降级处理 |
| 民办/高价项目 | 高 | 成本和认可度风险 | 单列，不进默认方案 |
| 省内偏好过窄 | 中 | 放弃更好专业/平台 | 加华东/华北对冲池 |
