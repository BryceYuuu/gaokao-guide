# Intake And Routing

Use this file when starting a gaokao guidance session, triaging incomplete information, or deciding which workflow to run.

For first contact or incomplete information, read `intake-form.md` first and send a copy-paste intake form with selectable options. Do not force families to invent preference language from scratch.

## Start With The Decision Map

Tell the user where they are:

1. 粗筛方向: information is incomplete; output low-confidence ranges and missing data.
2. 候选池: build broad school/major/group list from rank, rules, constraints, and preferences.
3. 深度核验: inspect specific schools, majors, professional groups, cost, campus, and risks.
4. 最终排表: produce ordered volunteer table with risk ledger and official-data refresh status.
5. 入学后兑现: plan transfer-major, GPA, graduate school, certifications, internships, and fallback paths.

## Required Inputs

Ask only for missing fields that materially change the answer.

| Field | Why it matters |
| --- | --- |
| 省份/年份 | Rules, score segments, plans, and volunteer units are province-year specific. |
| 批次/类型 | Ordinary, advance, arts, sports, special plans, directed programs, and vocational tracks use different constraints. |
| 科类/选科 | Determines eligibility and professional group limits. |
| 分数/位次 | Rank is the primary anchor; score alone is unstable across years. |
| 志愿单位 | School, school+major group, or major+school changes risk handling. |
| 学生偏好 | Interests, rejected subjects, ability, health, stress tolerance, and long-term path. |
| 家庭约束 | Budget, city, care needs, industry resources, risk tolerance, and acceptable training length. |
| 不可接受项 | Avoid false safety choices that the student would not actually attend. |

If rank is missing, ask for one of: official rank, one-score-one-segment row, full segment table, or a reliable third-party estimate. If the user only has estimated score, proceed as a draft and widen buffers.

## Intake Behavior

Use this order:

1. If the user has not provided the core fields, send the short intake form from `intake-form.md` and wait for the filled version.
2. If the user provided only score/rank and asks “能上什么”, send the short form plus say which two fields are mandatory to continue.
3. If the user filled most fields but left preferences blank, continue with conservative defaults and mark assumptions.
4. If the user asks for urgent rough direction, provide a rough answer, but still attach the missing-field checklist.

When presenting options, tell the user they can keep multiple choices, delete irrelevant options, or write their own.

Default to the medium copy-paste form. Use the full form only for final tables or deep reports. Use the ultra-short form only when the user wants speed over accuracy.

## Persona Tags

Use multiple tags when relevant:

- 高分宽选择: needs opportunity cost and overheat audit, not just preference satisfaction.
- 中上分段: compare school level, major strength, city, and graduate-school path.
- 中位窄选择: guard against overpaying for hot city/major; build wider counter-options.
- 本科压线: protect undergraduate admission and avoid unacceptable private/high-cost traps.
- 本专科临界: compare weak undergraduate options against strong vocational/industry routes.
- 专科/高职: focus on region, industry, training quality, transfer routes, and employability.
- 复读临界: compare repeat-year expected gain, psychological cost, family cost, and current admission floor.

## Ask Fewer, Infer Less

Do not invent values for vague goals. If a student says “想科研”“想进医院”“想稳定”“想高薪”, translate it into concrete paths and ask which path they mean.

Examples:

- “进医院” can mean clinician, medical technology, nursing, pharmacy, hospital administration, or public health.
- “计算机” can mean CS, software engineering, AI, data, cybersecurity, electronic information, automation, or vocational software track.
- “稳定” can mean civil service, teacher编制, medical system, state-owned enterprise, license-based profession, or local employment.

## Red Flags That Change Routing

- Only wanting one city or one hot major while rank is weak there: read strategy file and build counter-options.
- Willing to obey family against student ability or interest: mark as execution risk.
- High tuition, Chinese-foreign cooperation, private college, or independent college: require cost and credential audit.
- Professional group contains unacceptable majors: do group-level risk audit before ranking.
- Current-year data not published: label as simulation and create复核节点.
