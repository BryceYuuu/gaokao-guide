# Intake Form

Use this file at the beginning of a gaokao guidance session when the user has not provided enough information. Send a concise copy-paste form first; start analysis only after the user fills it or explicitly asks for a rough answer.

## Opening Line

Use this style:

> 我先收集关键信息。下面这张表可以直接复制粘贴填写；有选项的地方保留你认可的，删掉不符合的，也可以自己补充。填完后我再开始做位次、候选池、冲稳保和风险账本。

## Defaults If User Leaves Blanks

If the user leaves optional fields blank, use conservative defaults and state them:

- budget: public ordinary tuition preferred;
- risk: balanced but safety-aware;
- professional-group lower bound: unacceptable majors must be flagged;
- city: do not assume only local province;
- major: do not assume interest; offer several path families.

Do not assume willingness to accept private colleges, high tuition Chinese-foreign cooperation, remote campuses, long medical training, public-funded service obligations, or major adjustment into rejected fields.
